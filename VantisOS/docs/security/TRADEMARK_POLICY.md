## 1. The Philosophy
Vantis OS is open source, but the **"Vantis" brand** represents our reputation for security and quality. We must protect it to prevent phishing and malware distribution under our name.

## 2. Permitted Use (You don't need to ask)
* ✅ Using the logo to link to our official repository.
* ✅ Wearing Vantis-branded merchandise (t-shirts, stickers) for non-commercial use.
* ✅ Stating that your software "runs on Vantis OS".

## 3. Restricted Use (Requires permission)
* ❌ Selling a modified version of the OS called "Vantis OS Pro".
* ❌ Using the logo in a way that suggests you are a Vantis Corp partner.
* ❌ Changing the colors or proportions of the logo.

## 4. Forking Policy
If you fork the codebase to create your own OS, you **MUST** remove all Vantis trademarks, logos, and references to "Vantis Corp" from the UI assets.
