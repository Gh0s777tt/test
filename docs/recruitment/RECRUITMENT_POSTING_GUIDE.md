## 🎯 Interview Process

### Stage 1: Application Review (1 week)
- Review resume/CV
- Check GitHub profile
- Evaluate cover letter
- Assess technical fit

### Stage 2: Screening Call (30 min)
- Introduction and background
- Project overview
- Role expectations
- Availability and logistics

### Stage 3: Technical Interview (1-2 hours)
- Technical deep dive
- Problem-solving exercises
- Code review (if applicable)
- Architecture discussion

### Stage 4: Team Interview (1 hour)
- Meet team members
- Culture fit assessment
- Collaboration discussion
- Questions and answers

### Stage 5: Offer Discussion
- Compensation negotiation
- Start date coordination
- Onboarding planning
- Contract finalization
