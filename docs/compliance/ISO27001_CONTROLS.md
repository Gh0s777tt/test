## Control Framework Overview

### ISO/IEC 27001:2022 Control Structure

The 2022 version of ISO/IEC 27001 introduced a new control structure with 4 themes:

| Theme | Controls | Description |
|-------|----------|-------------|
| Organizational | 37 | Policies, roles, governance, and compliance |
| People | 8 | Human resources security and awareness |
| Physical | 14 | Physical security and environmental controls |
| Technological | 34 | Technical security controls |

### Control Attributes

Each control includes:

- **Control ID**: Unique identifier (e.g., A.5.1)
- **Control Name**: Descriptive name
- **Control Description**: Detailed description
- **Implementation Status**: Implemented, Partially Implemented, Not Implemented
- **Implementation Evidence**: Evidence of implementation
- **Compliance Score**: 0-100 score
- **Risk Level**: Low, Medium, High, Critical
