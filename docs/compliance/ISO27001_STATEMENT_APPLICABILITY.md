## SoA Overview

### Purpose of the SoA

The Statement of Applicability (SoA) is a critical document for ISO/IEC 27001 certification that:

1. **Documents Control Selection**: Identifies which controls are applicable to VantisOS
2. **Shows Implementation Status**: Demonstrates implementation of applicable controls
3. **Provides Justification**: Explains why controls are included or excluded
4. **Supports Certification**: Provides evidence for certification audits
5. **Enables Communication**: Communicates security posture to stakeholders

## Control Applicability Criteria

### Applicability Assessment

Controls are assessed for applicability based on:

1. **Relevance to VantisOS**: Control is relevant to VantisOS operations

3. **Business Requirements**: Control supports business requirements

### Key Control Justifications

#### Critical Controls

The following controls are considered critical for VantisOS:

1. **A.5.1.1 - Policies for information security**: Establishes security governance
2. **A.5.2.1 - Roles and responsibilities**: Defines accountability
