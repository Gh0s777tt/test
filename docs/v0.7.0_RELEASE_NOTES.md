# VantisOS v0.7.0 "IoT Ready" - Release Notes

## Overview

VantisOS v0.7.0 "IoT Ready" to wersja skupiona na wsparciu architektury RISC-V oraz urządzeń IoT. Jest to kluczowy krok w kierunku stworzenia uniwersalnego systemu operacyjnego dla urządzeń brzegowych i IoT.

## Release Date

**Planowane:** Kwiecień 2025

## Key Features

### 1. RISC-V Architecture Support ✅

VantisOS v0.7.0 wprowadza pełne wsparcie dla architektury RISC-V 64-bit (RV64GC):

#### Boot Process
- **Bootloader**: W pełni funkcjonalny bootloader RISC-V
- **Stack Setup**: Poprawne ustawienie stosu z 16-bajtowym wyrównaniem
- **BSS Clearing**: Automatyczne czyszczenie sekcji BSS
- **Hardware Initialization**: Inicjalizacja sprzętu RISC-V
- **MMU Setup**: Konfiguracja jednostki zarządzania pamięcią

#### Memory Management
- **Page Tables**: Implementacja tablic stron dla Sv39
- **Virtual Memory**: Pełne wsparcie dla pamięci wirtualnej
- **Memory Regions**: Zarządzanie regionami pamięci
- **Page Allocation**: Alokacja stron pamięci
- **Memory Protection**: Ochrona pamięci

#### Interrupt Handling
- **Trap Vector**: Wektor przerwań RISC-V
- **Exception Handling**: Obsługa wyjątków
- **PLIC**: Platform-Level Interrupt Controller
- **Timer Interrupts**: Przerwania zegarowe
- **System Calls**: Obsługa wywołań systemowych

#### Context Switching
- **Thread Switching**: Przełączanie wątków
- **Process Switching**: Przełączanie procesów
- **FPU State**: Zarządzanie stanem FPU
- **Context Save/Restore**: Zapisywanie i przywracanie kontekstu

#### SBI Interface
- **SBI Base**: Podstawowe funkcje SBI
- **SBI Timer**: Obsługa zegara SBI
- **SBI IPI**: Inter-Processor Interrupts
- **SBI RFENCE**: Remote Fence
- **SBI Console**: Konsola SBI
- **SBI SRST**: System Reset

#### Testing
- **Boot Tests**: Testy procesu rozruchu
- **MMU Tests**: Testy zarządzania pamięcią
- **Interrupt Tests**: Testy przerwań
- **Integration Tests**: Testy integracyjne

```json
  "features": "+m,+a,+f,+d,+c",
  "disable-redzone": true,
  "panic-strategy": "abort"

```

#### Memory Layout
```
Start            End              Size    Description
─────────────────────────────────────────────────────────
0x80000000       0x80010000       64KB    Boot code
0x80010000       0x80020000       64KB    Boot data
0x80020000       0x80200000       2MB     Kernel code
0x80200000       0x80400000       2MB     Kernel data
0x80400000       0x80800000       4MB     Kernel heap
0x80800000       0x81000000       8MB     User space
0x81000000       0x82000000       16MB    Device memory
0x82000000       0x88000000       96MB    Reserved
0x88000000       0x88010000       64KB    Stack
```

#### ISA Extensions
- **I**: Integer
- **M**: Multiply/Divide
- **A**: Atomic
- **F**: Single-Precision Floating-Point
- **D**: Double-Precision Floating-Point
- **C**: Compressed Instructions

### Build Instructions

#### Prerequisites
```bash
# Install RISC-V toolchain
rustup target add riscv64gc-unknown-none-elf

# Install QEMU RISC-V
apt-get install qemu-system-misc
```

#### Building
```bash
# Build RISC-V kernel
./build_riscv_kernel.sh

# Run on QEMU
qemu-system-riscv64 \
  -machine virt \
  -nographic \
  -bios none \
  -kernel target/riscv64gc-unknown-none-elf/release/vantisos \
  -m 128M
```

3. Zbuduj kernel RISC-V
